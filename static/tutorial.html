Finance Tracker - How It Works

Welcome to Finance Tracker! This comprehensive personal finance management system helps you track expenses, manage budgets, and optimize your financial planning with smart allocation features.
1

Authentication & Setup

Start by registering your account and logging in to access the full feature set.

POST /auth/register
POST /auth/login
2

Manage Categories

Create and organize your expense categories for better transaction organization.

POST /categories
GET /categories
PUT /categories/{id}
3

Track Transactions

Add individual transactions or bulk upload your financial data. Support for both single and bulk operations.

POST /transactions (single or bulk)
PUT /transactions (single or bulk)
POST /transactions/list
4

Set Up Budgets

Create detailed budgets with category-specific limits for better financial control.

POST /budgets
GET /budgets?year_month=2024-03
GET /budgets/comparison/monthly
5

Recurring Transactions

Set up recurring income and expenses with advanced scheduling options including date flexibility and priority levels.

POST /recurring-transactions
GET /recurring-transactions
PUT /recurring-transactions/{id}/status
6

Smart Allocation

Use our intelligent allocation system to optimize your budget based on income patterns and upcoming expenses.

GET /allocation?year_month=2024-03
7

Dashboard & Analytics

View comprehensive financial insights with month-by-month analysis and expense categorization.

GET /dashboard?year_month=2024-03
GET /dashboard/expenses-by-category
GET /expenses
8

Reports & Export

Generate detailed budget reports in CSV format for external analysis.

GET /budgets/report?year=2024&month=3&format=csv

Key Features

  • Enhanced Budget System: Category-specific budgets with spending tracking
  • Smart Allocation: AI-powered budget optimization based on spending patterns
  • Advanced Recurring Transactions: Flexible scheduling with date flexibility options
  • Bulk Operations: Efficient bulk transaction processing
  • Month-based Analytics: Detailed financial insights by time period
  • Export Capabilities: CSV reports for external analysis
  • RESTful API: Complete programmatic access to all features

API Documentation

For complete API documentation, visit /docs for the interactive Swagger UI or /redoc for the ReDoc interface.

Getting Started: Begin by creating categories for your expenses, then add some transactions, and set up your first budget. The system will learn from your patterns and provide intelligent allocation recommendations.
